## Compilation :

go build



## Flags:

-file   :   Path du fichier en entrée (-file='test.txt')

-o      :   Path du fichier de sortie (-o='test.out')

-h      :   Affiche l'aide



## Exécution:

./Compilateur.exe -file='test.txt' -o='test.out'    (Windows)

./Compilateur -file='test.txt' -o='test.out'    (Linux)
